## What I'm building

**[NexusQuant](https://nexusquant.co/)** is a cloud-based quantitative trading platform that lets traders deploy and run algorithmic strategies across multiple brokers — Moomoo, Futu, Tiger, Webull, Charles Schwab, and Interactive Brokers — through a unified interface. The platform manages containerized trading agents on AWS, routes signals in real time via Redis Pub/Sub, and handles the full broker connectivity lifecycle including OAuth flows, credential encryption with AWS KMS, and token renewal for brokers like Schwab and IBKR that require periodic re-authentication.

**[Makea](https://www.makea.co/)** is a garment manufacturing platform that connects clothing designers with suppliers through the full production lifecycle — from initial product specs and sampling requests to production orders, supplier matching, and document management. Built on AWS with a DynamoDB single-table design, S3 for file storage, and real-time collaboration over WebSockets, it streamlines a workflow that has traditionally relied on emails and spreadsheets.
